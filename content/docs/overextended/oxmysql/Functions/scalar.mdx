---
title: scalar
---

Retorna a primeira coluna de uma única linha.

## Promise

<Tabs items={['Lua', 'JS']}>
  <Tabs.Tab>

    ```lua
    local firstName = MySQL.scalar.await('SELECT `firstname` FROM `users` WHERE `identifier` = ? LIMIT 1', {
        identifier
    })

    print(firstName)
    ```

  </Tabs.Tab>
  <Tabs.Tab>

    ```js
    const firstName = await MySQL.scalar('SELECT `firstname` FROM `users` WHERE `identifier` = ? LIMIT 1', [
      identifier
    ])

    console.log(firstName)
    ```

  </Tabs.Tab>
</Tabs>

**Aliases**

- `MySQL.Sync.fetchScalar`
- `exports.ghmattimysql.scalar`
- `exports.oxmysql.scalar_async`

## Callback

<Tabs items={['Lua', 'JS']}>
  <Tabs.Tab>

    ```lua
    MySQL.scalar('SELECT `firstname` FROM `users` WHERE `identifier` = ? LIMIT 1', {
        identifier
    }, function(firstName)
        print(firstName)
    end)
    ```

  </Tabs.Tab>
  <Tabs.Tab>

    ```js
    MySQL.scalar('SELECT `firstname` FROM `users` WHERE `identifier` = ? LIMIT 1', [
      identifier
    ], (firstName) => {
      console.log(firstName)
    })
    ```

  </Tabs.Tab>
</Tabs>

**Aliases**

- `MySQL.Async.fetchScalar`
- `exports.ghmattimysql.scalar`
- `exports.oxmysql.scalar`
